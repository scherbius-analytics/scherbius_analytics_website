Quantitative Strategie

Scherbius 1.0 Institutionell ist eine algorithmische Handelsstrategie, die auf regelbasierter Signalverarbeitung und dynamischem Risikomanagement basiert. Das Ziel: robuste, marktunkorrelierte Renditen mit streng kontrolliertem Drawdown.

Scherbius 1.0 Privatanleger ist eine algorithmische Handelsstrategie, die auf regelbasierter Signalverarbeitung basiert. Das Ziel: überlegene langfristige Renditen durch systematische Ausnutzung von Marktineffizienzen.

Kernmethodik

Vier Säulen der
Scherbius-Methode

Regelbasierte Signalerzeugung

Sämtliche Kauf- und Verkaufssignale werden durch deterministische Algorithmen erzeugt. Es gibt keine diskretionären Eingriffe, keine menschliche Übersteuerung. Das System berechnet täglich auf Basis aktueller Marktdaten die optimalen Positionen für jedes Instrument im Portfolio.

Dynamische Positionsgrößen

Die Größe jeder Position wird täglich neu berechnet und an das aktuelle Marktregime angepasst. In Phasen erhöhter Volatilität reduziert das System Positionen proaktiv, bevor Verluste entstehen — nicht reaktiv danach.

Korrelationsmanagement

Das Portfolio ist so konstruiert, dass es einen minimalen Beta-Koeffizienten zum Gesamtmarkt aufweist. Durch die gleichzeitige Haltung von Long- und Short-Positionen werden Marktphasen-Risiken strukturell reduziert. Das Ergebnis: ein Beta von 0,127 gegenüber dem NASDAQ-100.

Drawdown-Kontrolle

Ein mehrstufiges Drawdown-Kontrollsystem überwacht das Portfolio in Echtzeit. Sobald definierte Schwellenwerte erreicht werden, reduziert das System automatisch die Gesamtexponierung — unabhängig von Markterwartungen oder externen Meinungen.

Risiko-Framework

Risiko ist keine Nebenbedingung.
Es ist das Kernproblem.

Die meisten Anlagestrategien optimieren zuerst auf Rendite und behandeln Risiko als Constraint. Scherbius 1.0 dreht diese Logik um: Risiko wird zuerst definiert und kontrolliert, Rendite entsteht als Konsequenz disziplinierter Risikonahme.
